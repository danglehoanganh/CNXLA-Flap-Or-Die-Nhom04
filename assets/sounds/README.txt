# Hướng dẫn thêm âm thanh (tiếng Việt)

**Nhạc nền (background music):**
- Đặt file MP3 của bạn vào đây với tên: `bg_music.mp3`
- Hỗ trợ: MP3, OGG, WAV
- Nhạc sẽ tự động loop khi game bắt đầu

**Âm thanh qua chướng ngại vật (pass obstacle):**
- Đặt file WAV/MP3 của bạn vào đây với tên: `point.wav`
- Âm thanh phát tự động mỗi khi chim bay qua ống (score +1)

**Lưu ý:**
- Thay tên file trong code game.py dòng ~430 nếu dùng tên khác
- Format tốt nhất: WAV 44.1kHz cho effect, MP3 cho music
- Không có file? Code vẫn chạy bình thường (silent mode)

Thay file và chạy game để nghe!
